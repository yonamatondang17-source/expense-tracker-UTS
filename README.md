# 💰 Expense Tracker - UTS React

Aplikasi pencatat transaksi keuangan pribadi berbasis React + Vite.

## 📸 Screenshot

![Tampilan Expense Tracker](./Screenshot 2026-05-13 121035.png)

## ✨ Fitur

- ✅ Header saldo otomatis berubah
- ✅ Tambah pemasukan & pengeluaran
- ✅ Nominal hijau (pemasukan) & merah (pengeluaran)
- ✅ Riwayat transaksi lengkap

## 🛠️ Teknologi

- React 19
- Vite
- CSS3

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
